HSL color scheme

  • Hue: the color is determined by the hue value as represented in 360 degrees of the HSL color wheel.
  • Saturation: ranges from 0 - 100. Zero being completely desaturated with 100 representing the full saturation of your hue.
  • Lightness: also ranges from 0 - 100 with 0 denoting black while 100 will return white.
